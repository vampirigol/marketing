## Guía rápida (primeros pasos)

1. **Entrar al sistema:** Abre la URL del CRM e inicia sesión en `/login` con tu usuario y contraseña (las entrega el administrador).
2. **Elegir sucursal:** En el menú lateral, debajo de los módulos, selecciona tu sucursal para que Recepción, Citas y Calendario muestren datos correctos.
3. **Recepción (llegadas):** Ve a **Recepción** → localiza la cita → **Marcar Llegada** → después **Pasar a Consultorio** → al terminar **Finalizar**. Si no llega, **Inasistencia**.
4. **Agendar una cita:** Ve a **Citas** → **Nueva cita** (o clic en slot vacío) → elige paciente, sucursal, doctor, fecha y hora → guardar.
5. **Ver un paciente:** **Pacientes** → buscar → clic en el ojo (Ver) para abrir el **Expediente** con datos, citas y pagos.
6. **Contact center:** **Keila IA** → Inbox para conversaciones; Kanban para leads. Desde una conversación puedes **Agendar cita** o cambiar prioridad y etiquetas.

## Cómo usar este manual

Este manual describe el **funcionamiento completo** del sistema CRM RCA: cada pantalla, botón, flujo y automatización, para que cualquier usuario entienda cómo usar el sistema.

- **Si eres nuevo:** empieza por [01 - Visión general](01-VISION-GENERAL.md).
- **Si buscas un módulo concreto:** ve a [02 - Módulos](02-MODULOS/) y abre el archivo del módulo (ej. Recepción, Citas, CRM).
- **Si quieres entender las automatizaciones:** consulta [03 - Automatizaciones y flujos](03-AUTOMATIZACIONES-Y-FLUJOS.md).
- **Si necesitas el flujo completo de un proceso:** revisa [04 - Procesos de negocio](04-PROCESOS-DE-NEGOCIO.md).
- **Si no entiendes un término:** revisa [05 - Glosario y referencia](05-GLOSARIO-Y-REFERENCIA.md).
- **Login, perfil, reservar, confirmar cita, automatizaciones:** ver [06 - Páginas y rutas adicionales](06-PAGINAS-Y-RUTAS-ADICIONALES.md).
- **Diagramas de flujo (leads, citas, recepción, conversión):** ver [07 - Diagramas de flujo](07-DIAGRAMAS-DE-FLUJO.md).
