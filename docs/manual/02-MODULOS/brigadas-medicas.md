# Módulo Brigadas Médicas

## Descripción

Gestión de **brigadas médicas**: jornadas o campañas de salud en las que se registran atenciones (pacientes atendidos, rango de edad, servicios). Incluye listado de brigadas, registro de atenciones, resumen por brigada (total atendidos, KPIs) y reportes para análisis.

**Ruta:** Brigadas Médicas → `/brigadas-medicas`

---

## Pantalla principal

- **Listado o selector de brigadas** (nombre, fecha, ubicación, estado).
- **Resumen de la brigada activa:** Total de pacientes atendidos, rango de edad, servicios más demandados.
- **Registro de atenciones:** Formulario o tabla para cargar atenciones (paciente, edad, servicio, etc.).
- **Reportes o exportación** (si está disponible).

---

## Botones y acciones

| Elemento | Acción |
|----------|--------|
| **Nueva brigada** | Crear una nueva brigada (nombre, fechas, sede). |
| **Registrar atención** | Añadir registro de paciente atendido (edad, servicio, notas). |
| **Editar brigada** | Modificar datos de la brigada seleccionada. |
| **Ver reporte** | Resumen o exportación (total atendidos, por edad, por servicio). |

---

## Flujos típicos

- Crear brigada para una jornada.
- Durante la jornada, registrar cada atención.
- Al cierre, revisar total atendidos y rango de edad; exportar o imprimir reporte si aplica.
