# App movil (mobile)

Esta carpeta contiene la aplicacion movil del sistema.

Decisiones iniciales:
- Tecnologia: React Native con Expo
- Carpeta: mobile/

Como correr:
1) cd mobile
2) npm run ios | npm run android | npm run web

Siguientes pasos sugeridos:
1) Configurar entorno (dev/staging/prod)
2) Conectar con el backend actual (auth y endpoints clave)
3) Definir modulos iniciales (citas, pacientes, recepcion, matrix)

Nombres del sistema:
- CRM Web: aplicacion web (frontend en navegador)
- API CRM: backend / API (endpoints y datos)
- RCA Mobile: aplicacion movil

Notas:
- Si prefieres Flutter o un repo separado, lo ajustamos.
